# Cortex-M3 bare metal programming

## Setup

- Install [QEMU](https://www.qemu.org/) for ARM Cortex-M3 simulation.
- Install [ARM GNU Toolchain](https://developer.arm.com/downloads/-/gnu-a/11.3.rel1/downloads) for Cortex-M3 cross-compilation.
- Install [Cortex-Debug](https://marketplace.visualstudio.com/items?itemName=marus25.cortex-debug) extension for VSCode.
